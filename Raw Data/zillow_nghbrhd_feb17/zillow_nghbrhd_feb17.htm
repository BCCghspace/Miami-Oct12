DATA LINEAGE SUMMARY: 








This dataset was downloaded by the GeoPlan Center from the 
Zillow website (https://www.zillow.com/howto/api/neighborhood-boundaries.htm
on 02/24/2017. When received the data set was in the following projection: 
Albers NAD 83. The data was reprojected to Albers NAD 83 HARN.

- Deleted two features that were incorrectly attributed to the state of Florida. 
  Features were REGIONID 698707 and 737815.
- Added the field DESCRIPT based on NAME.
- Added the field FGDLAQDATE based on date downloaded from source.
- Upcased all text in the attribute table.
- Data set was renamed zillow_nghbrhd_feb17 from ZillowNeighborhoods-FL
Process Date: 20170224
